# Weather App

A simple weather application that provides current weather information based on the city entered by the user.

## Features
- Search for weather by city name
- Displays temperature, humidity, and wind speed
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
